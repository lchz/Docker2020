# Docker for reading tip-miniproject
This application is built for saving and searching reading tips. A reading tip is consisted of a title, an author, some description, a link and a bunch of tags. Reading tips can be searched based on the titles or the tags.


## Manual for local running
The application is built with Java 11 and Gradle 6.3. Use command `./gradlew run` to start it locally at [http://localhost:4566](http://localhost:4566).
